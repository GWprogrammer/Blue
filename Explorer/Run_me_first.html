Blue Explorer

Blue is 100% independent, it will not use nor modify any of your existing software.

Blue will only Run on Olibasic III and the enclosed Version of the GW lib file.

Blue.zip and Blue_Setup.bas

Blue.zip should be in the folder Downloads/

Blue_Setup.bas Should be in Source/, Run this file first, it will unzip blue and put all files in proper folders.

Then simply run the file: Blue.bas

All Icons are copied to Assets: storage/Blue/Icons/
